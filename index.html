The Random numbers Game

Como Jogar?

  1. Escolha um número de 0 a 1000 e clique em "Chutar";
  2. O pato irá lhe dizer se o número que você forneceu é maior ou menor que o número dele;
  3. Caso seja maior, você deve tentar um número maior ao que você disse anteriormente e, caso seja menor, deve informar um número menor;
  4. Repita o 3 passo até acertar o número que o pato escolheu.

Seja bem-vindo(a)! Para começar, escolha um número entre 0 e 1000.
